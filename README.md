# -Copilot-Modes-Guia-de-Refer-ncia
Guia definitivo dos Modos do Copiloto (Ask, Edit, Plan, Study). Uma documentação estruturada para desenvolvedores que desejam dominar a inteligência artificial no fluxo de trabalho, diferenciando consulta técnica, transformação de código, planejamento arquitetural e aprendizado ativo.
